API Access

Demo Endpoints (no key required)

POST/demo/session/create — start a demo engine session
POST/demo/session/{id}/scan — advance one synthetic window
POST/demo/validate/all — run full verification harness (public)
GET/status — engine health check
Demo endpoints are rate-limited. Synthetic waveforms only — no real data ingestion.

Trial Endpoints (trial code required)

POST/trial/session/create
POST/trial/session/{id}/push
GET/trial/session/{id}/status
DELETE/trial/session/{id}
POST/trial/csv/{domain}
Trial endpoints accept real waveform data. Contact AW IP Holdings Inc. for a trial code.

Sample — Create Trial Session

curl -X POST https://tc-domain-api.onrender.com/trial/session/create \
  -H "Content-Type: application/json" \
  -d '{"domain":"seismic","trial_code":"YOUR-TRIAL-CODE","fs":100}'

Sample — Push Waveform Window

curl -X POST https://tc-domain-api.onrender.com/trial/session/{session_id}/push \
  -H "Content-Type: application/json" \
  -d '{
    "trial_code": "YOUR-TRIAL-CODE",
    "waveform": [0.012, -0.034, 0.051, -0.008, 0.022, ...],
    "mc": 0.0
  }'
Waveform must be normalized to [-1, 1]. Minimum 16 samples, maximum 20,000.
